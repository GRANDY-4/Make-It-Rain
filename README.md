# Make It Rain (MIR) - Python Edition

A Payday-like heist game built with **Python and Pygame**.

## Quick Start

### Prerequisites
- **Python 3.8+**
- **pip** (Python package manager)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/GRANDY-4/Make-It-Rain.git
cd Make-It-Rain
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the game:
```bash
python main.py
```

## Controls

- **WASD** — Move player
- **P** — Pause/Resume
- **ESC** — Exit game
